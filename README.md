# Water Tank Monitor

An automated water tank monitoring system that was  built using  Python.

## What it does
- Simulates a water tank filling cycle automatically
- Monitors water level every 10 seconds
- Sends real email alerts when tank is full or water is low
- Logs all readings to a file

## Technologies used
- Python
- yagmail — email notifications
- schedule — automated monitoring
- File handling — data logging

## Skills demonstrated
- Automation and scripting
- API integration
- Scheduled tasks
- Real world system monitoring
